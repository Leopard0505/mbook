## How to use it?
1. ローカルにmysqlを立ち上げる
    1. ポート番号は`3306`
    2. データベース名は`mbook`
    3. テーブルは`server/src/test/resources/schema.sql`
    4. データは`server/src/test/resources/data.sql`
2. ターミナルでこのプロジェクトのルートディレクトリに移動する
3. `server`ディレクトリに移動し、バックエンドを立ち上げる
4. `web`ディレクトリに移動し、フロントエンドを立ち上げる
